TikTok Publishing Automation

This web service demonstrates an automation workflow used to schedule and publish short videos to TikTok.

Content data is prepared through n8n, including video URL, thumbnail URL, caption text, hashtags, full caption, row reference, and publishing status before sending the content to TikTok through API integration after authorization.

Workflow Overview

This workflow reads content data from a spreadsheet, retrieves the video and thumbnail, formats the publishing fields in n8n, and prepares the content for TikTok publishing.
